# Museos de Ecuador

Sitio web informativo sobre los principales museos del Ecuador, con mapa interactivo y galería de imágenes.

## Tecnologías utilizadas
- HTML5
- CSS (Tailwind CSS)
- JavaScript vanilla
- Leaflet.js (mapas)
